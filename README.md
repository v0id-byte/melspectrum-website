# MelSpectrum 公司官网

> 融谱智能科技 — 让机器听懂声音

静态官网（HTML + CSS + Vanilla JS），展示公司及旗下产品（Piano Tuner / Somnil）。

---

## 技术栈

- **前端**: 原生 HTML5 + CSS3 + JavaScript（无框架）
- **字体**: Inter（Google Fonts）
- **部署**: GitHub Pages + CNAME（melspectrum.com）

---

## 功能列表

- 🏢 **公司介绍**：声学智能科技定位、团队介绍
- 🎹 **产品展示**：Piano Tuner 自动钢琴调律机器人
- 😴 **产品展示**：Somnil 睡眠焦虑检测设备
- 📧 **联系方式**：somnil@melspectrum.com

---

## 安装说明

```bash
# 克隆项目
git clone https://github.com/v0id-byte/melspectrum-website.git
cd melspectrum-website

# 本地预览
python3 -m http.server 8080
# 访问 http://localhost:8080
```

> 部署：推送到 GitHub Pages 分支，配置 CNAME `melspectrum.com`。
