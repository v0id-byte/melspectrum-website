01 · 入场动画
Hero 文字交错渐入
页面加载时，badge → 主标题 → 副标题 → 按钮依次淡入上浮，间隔 150ms。 cubic-bezier(0.16, 1, 0.3, 1) 带来轻微的弹性感，适合首屏。
/* 核心动画 keyframes */ @keyframes fadeInUp { from { opacity: 0; transform: translateY(24px); } to { opacity: 1; transform: translateY(0); } } @keyframes fadeInDown { from { opacity: 0; transform: translateY(-16px); } to { opacity: 1; transform: translateY(0); } } /* 各元素错开延迟 */ .hero-badge { animation: fadeInDown 0.7s ... 0.1s forwards; } .hero-title { animation: fadeInUp 0.7s ... 0.25s forwards; } .hero-subtitle { animation: fadeInUp 0.7s ... 0.55s forwards; } .hero-cta { animation: fadeInUp 0.7s ... 0.7s forwards; }

/* Primary 扫光 */ .btn-primary::before { content: ''; position: absolute; top: 0; left: -100%; width: 100%; height: 100%; background: linear-gradient(90deg, transparent, rgba(255,255,255,0.35), transparent); transition: left 0.55s ease; z-index: 1; } .btn-primary:hover::before { left: 100%; } /* Ghost 渐变填充 */ .btn-ghost::before { content: ''; position: absolute; inset: -1.5px; border-radius: 50px; background: linear-gradient(135deg, var(--accent), var(--accent-light)); opacity: 0; transition: opacity 0.25s ease; z-index: -1; } .btn-ghost:hover { border-color: transparent; color: #fff; } .btn-ghost:hover::before { opacity: 1; }

06 · 滚动动画
滚动触发的渐入动画
Intersection Observer API 监听元素进入视口，批量添加 .revealed 类。 配合 transition-delay 实现错落有致的卡片入场效果。
const observer = new IntersectionObserver( entries => entries.forEach(entry => { if (entry.isIntersecting) { entry.target.classList.add('revealed'); observer.unobserve(entry.target); } }), { threshold: 0.15 } ); document.querySelectorAll('.reveal-card').forEach(el => observer.observe(el)); /* CSS */ .reveal-card { opacity: 0; transform: translateY(32px); transition: opacity 0.65s cubic-bezier(0.16,1,0.3,1), transform 0.65s cubic-bezier(0.16,1,0.3,1); } .reveal-card.revealed { opacity: 1; transform: translateY(0); } .reveal-card:nth-child(2) { transition-delay: 0.1s; } .reveal-card:nth-child(3) { transition-delay: 0.2s; } .reveal-card:nth-child(4) { transition-delay: 0.3s; }
